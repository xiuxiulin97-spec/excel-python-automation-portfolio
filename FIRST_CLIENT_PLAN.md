# 30 天找到第一個 Excel 自動化客戶計劃

## 目標

在 30 天內找到第一個願意付費的 Excel / 資料整理 / Google Sheets 自動化客戶，完成一個小型可交付專案，累積第一個真實案例。

## 1. 目前可展示作品

### Excel Report Tool

可展示能力：

- 讀取 Excel
- 清除空白列
- 刪除重複資料
- 依客戶統計金額
- 產生 Excel 報表
- 打包成 Windows EXE

適合展示給：每月需要整理銷售報表的小公司、工作室、業務團隊。

### Customer Cleaner

可展示能力：

- 支援 Excel / CSV
- 統一欄位名稱
- 清理電話格式
- 檢查 Email
- 刪除重複客戶
- 輸出乾淨名單與問題資料

適合展示給：活動主辦方、補習班、電商、業務團隊、名單行銷團隊。

### Google Sheets Report Sync

可展示能力：

- 讀取 Excel / CSV
- 清理資料
- 產生統計報表
- 使用 Google Sheets API
- OAuth 授權
- 同步到 Google Sheets

適合展示給：遠端團隊、小型公司、需要多人查看資料的工作室。

## 2. 最適合接的小型案件

第一個案子不要接太大，優先選擇 3-7 天能完成的工作。

適合案件：

- 每月銷售 Excel 自動整理
- 活動報名名單清洗
- 客戶資料去重與 Email 檢查
- CSV 轉成固定 Excel 報表格式
- Excel 報表自動統計
- Excel 清理後同步 Google Sheets

暫時不要接：

- 大型 CRM 系統
- ERP 串接
- 多人權限管理
- 複雜排程系統
- 需要長期駐場的案子

## 3. 如何介紹服務

### 對朋友介紹

可以這樣說：

```text
我最近在做 Excel 自動化工具，主要幫小公司把重複的 Excel 整理、客戶名單清洗、報表產生變成按鈕式工具。

如果你或朋友每個月都要手動整理 Excel、複製貼上、做固定報表，我可以幫忙做一個小工具，讓流程自動化。
```

### 對工作室介紹

可以這樣說：

```text
我可以協助把活動名單、客戶資料、銷售 Excel 做自動清理與報表輸出。

例如清除重複資料、檢查 Email、統計客戶金額、輸出乾淨 Excel，或同步到 Google Sheets，讓團隊不用一直手動整理。
```

### 對小公司介紹

可以這樣說：

```text
我協助中小企業把日常 Excel 報表流程自動化。

如果你們有固定格式的 Excel / CSV，例如銷售表、客戶名單、活動報名表，我可以做成一個 Windows 小工具，讓同事選檔後自動產生整理好的報表。
```

## 4. 第一個案子建議報價

第一個付費案的目標是取得真實案例與客戶回饋，不是一次報到最高價。

建議報價：

| 案件類型 | 建議價格 |
| --- | ---: |
| 簡單 Excel 清理 | NT$3,000-6,000 |
| Excel 自動報表 | NT$6,000-12,000 |
| 客戶名單清洗工具 | NT$5,000-10,000 |
| Excel + Google Sheets 同步 | NT$12,000-25,000 |

第一個案子建議主推：

```text
NT$5,000-12,000 的小型 Excel 自動化工具
```

## 5. 第一個案子接案流程

1. 簡單了解客戶目前怎麼整理資料
2. 請客戶提供一份去識別化範例檔
3. 確認輸入欄位與輸出格式
4. 寫一頁需求確認
5. 報價與確認交付時間
6. 收訂金
7. 製作第一版
8. 用範例資料測試
9. 交付 EXE 與使用說明
10. 收尾款
11. 收集回饋與案例截圖

## 6. 如何收集需求

請客戶回答以下問題：

1. 你現在的 Excel / CSV 是從哪裡匯出的？
2. 每次整理大約花多久？
3. 哪些步驟最重複、最容易出錯？
4. 原始檔有哪些欄位？
5. 最後想產生什麼結果？
6. 需要 Excel 輸出、Google Sheets，還是兩者都要？
7. 這個流程多久做一次？
8. 誰會使用這個工具？
9. 是否有個資或敏感資料？
10. 是否可以提供測試用範例檔？

需求確認範例：

```text
輸入：銷售 Excel
必要欄位：客戶、金額、日期
處理規則：刪除空白列、刪除重複資料、依客戶統計金額
輸出：result.xlsx
交付：Windows EXE + README
修改：包含 1 次小幅修改
```

## 7. 如何交付 EXE 工具

交付內容建議包含：

- `ExcelReportTool.exe`
- `README.md`
- `sample.xlsx`
- `使用說明.pdf` 或簡單文字說明
- 範例輸出結果

交付方式：

1. 建立一個交付資料夾
2. 放入 EXE 與說明文件
3. 不放客戶正式資料
4. 用範例資料示範操作
5. 壓縮成 zip
6. 交給客戶測試

交付提醒：

- 不要把客戶資料放進 GitHub
- 不要把客戶資料打包進 EXE
- 客戶正式資料只留在客戶端使用
- 如果工具會產出報表，預設輸出到 `output/`

## 8. 如何收取修改需求

第一個案子建議包含：

- 1 次小幅修改
- 修改時間限交付後 7 天內
- 超出範圍另行報價

小幅修改範例：

- 調整欄位名稱
- 修改輸出檔名
- 新增一個簡單統計欄位
- 調整錯誤提示文字

超出範圍範例：

- 新增 Google Sheets 同步
- 新增多個工作表合併
- 新增自動排程
- 新增登入權限
- 改成完整系統

可以這樣說：

```text
本次報價包含 1 次小幅修改。如果後續需要新增功能，我會先整理需求並另外報價，確認後再開始製作。
```

## 9. 如何累積第一個案例

完成第一個案子後，請客戶允許你保留去識別化案例。

要收集的素材：

- 原本流程花多少時間
- 工具完成後節省多少時間
- 工具截圖
- 輸入與輸出前後對比
- 客戶一句回饋
- 是否可匿名寫成案例

案例格式：

```text
案例名稱：每月銷售報表自動化
客戶類型：小型工作室
問題：每月手動整理 Excel 約 2 小時
解決方案：建立 Windows EXE 工具，自動清理資料並產生統計報表
成果：整理時間縮短為 5 分鐘內
```
