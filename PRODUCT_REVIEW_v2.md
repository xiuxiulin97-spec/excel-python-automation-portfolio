# Excel Report Tool v2.0 / v2.1 產品審查

本文件根據目前專案實際狀態撰寫，不列入尚未完成的功能作為已完成項目。

## 1. 目前已完成功能

### v2.0 已完成

Excel Report Tool v2.0 已從固定欄位工具，升級為通用表格分析工具。

已完成：

- 支援讀取 Excel / CSV
- Excel 固定讀取第一個工作表
- 清除全空白列
- 刪除重複資料
- 自動分析欄位類型
- 判斷文字欄位、數字欄位、日期欄位
- GUI 顯示：
  - 檔名
  - 列數
  - 欄位數
  - 欄位分析結果
- 使用者可自行選擇：
  - 分組欄位
  - 統計欄位
  - 日期欄位，可選「不使用」
- 可產生 Excel 報表
- 輸出工作表包含：
  - 原始資料
  - 清理後資料
  - 統計報表
- 統計報表包含：
  - 總覽
  - 分組統計
  - 月份統計，當使用日期欄位時
  - Top10 排行
- 金額欄位可處理常見格式，例如 `元`、`￥`、`NT$`、逗號與空白值

### v2.1 已完成

v2.1 目前完成的是 Excel 輸出美化，不新增分析邏輯。

## 2. 哪些功能最有商業價值

最有商業價值的是這幾個：

- 不固定欄位名稱
- 使用者可自行選擇分組欄位與統計欄位
- 自動清理空白列與重複資料
- 自動產生統計報表
- 自動產生 Top10 排行
- 可處理常見金額格式
- 輸出 Excel 已有正式表格樣式

對中小企業來說，這些功能能直接解決：

- 每次都要手動整理 Excel
- 每個客戶或部門的欄位名稱不同
- 報表需要人工複製貼上
- 金額統計容易出錯
- 老闆或主管希望快速看到分類統計與排行

v2.1 的表格美化也有商業價值，因為客戶打開報表時會先看外觀。如果輸出的 Excel 像正式報表，工具會更容易被接受。

## 3. 哪些功能其實很少人會用

以下功能有價值，但不是每個客戶都會常用：

- 日期欄位分析
- 月份統計
- Top10 排行
- 欄位類型分析表

原因：

- 很多客戶只想要「照某欄分類，加總某欄金額」
- 有些表格沒有日期欄位
- 有些報表不需要 Top10
- 一般客戶不一定理解欄位類型分析，只關心結果是否產出

這些功能可以保留，但對外展示時不應講得太複雜。銷售時應該強調：

選 Excel、選欄位、按一下，產生整理好的報表。

## 5. 哪些功能應該延後到 v3.0

以下功能應延後到 v3.0：

- AI 摘要
- PDF 匯出
- 圖表
- 儀表板
- 多工作表合併
- 多檔案批次處理
- 自動排程
- 客戶自訂報表模板
- EXE 安裝器
- 權限或帳號管理

原因：

目前 v2.1 的價值是穩定產生通用 Excel 報表。若太早加入 AI、PDF、圖表或自動排程，會增加複雜度，也會讓交付風險變高。

## 6. 如果我是客戶

### 最喜歡什麼

如果我是客戶，我最喜歡：

- 不需要懂 Python
- 不需要寫 Excel 公式
- 不需要重新整理固定格式
- 可以使用自己的 Excel / CSV
- 可以自行選擇分組欄位與統計欄位
- 輸出檔包含原始資料、清理後資料與統計報表
- 輸出表格已經有正式樣式
- 報表可以直接打開檢查或交給主管看

最有感的商業價值是：

原本需要人工整理的 Excel，可以快速變成一份可閱讀、可篩選、可統計的報表。

### 最容易卡在哪裡

客戶最容易卡在：

- 不知道該選哪個分組欄位
- 不知道該選哪個統計欄位
- 原始 Excel 標題列太亂，造成欄位辨識不準
- 金額欄混入太多文字，統計結果需要人工確認
- 日期格式太混亂，月份統計可能不符合預期
- 不知道輸出檔在哪裡

後續改善方向：

- GUI 加入欄位推薦提示
- 錯誤訊息更白話
- README 補完整操作截圖
- 客戶版使用說明寫清楚輸出位置
