# 服務項目

我提供 Excel、客戶資料與 Google Sheets 相關的自動化工具，協助中小企業與個人工作室減少重複整理資料的時間。

## 1. Excel 自動報表工具

### 問題類型

- 每月或每週都要手動整理 Excel
- 報表需要重複複製、貼上、加總
- 空白列、重複資料與公式錯誤造成時間浪費

### 解決方案

建立可執行的 Excel 自動報表工具，自動讀取資料、清理內容、統計金額並輸出報表。

### 交付方式

- Windows EXE 工具
- Python 原始碼
- README 使用說明
- 範例輸入與輸出檔案

### 基礎價格區間

NT$6,000 - NT$15,000

### 適合客戶

- 小型工作室
- 業務團隊
- 行政人員
- 每月需要固定報表的中小企業

## 2. 客戶資料清洗工具

### 問題類型

- 客戶名單有重複資料
- Email 格式錯誤
- 電話格式不一致
- 欄位名稱混亂，難以匯入其他系統

### 解決方案

建立客戶資料清洗工具，自動整理欄位、清除重複資料、檢查 Email、清理電話格式，並輸出乾淨名單與問題資料。

### 交付方式

- Windows EXE 工具或 Python 專案
- 清理後 Excel 檔案
- 問題資料清單
- README 使用說明

### 基礎價格區間

NT$5,000 - NT$12,000

### 適合客戶

- 活動主辦方
- 補習班
- 電商賣家
- 業務團隊
- 需要整理客戶名單的工作室

## 3. Google Sheets 自動同步

### 問題類型

- 多人使用不同版本的 Excel
- 手動把資料貼到 Google Sheets
- 團隊無法即時查看最新報表
- 雲端資料與本機檔案不同步

### 解決方案

建立 Google Sheets 自動同步工具，將 Excel / CSV 清理後，自動寫入指定 Google Sheets，產生清理後資料與統計報表。

### 交付方式

- Python 桌面工具
- Google Sheets API OAuth 設定說明
- README 使用說明
- 測試資料與範例輸出

### 基礎價格區間

NT$12,000 - NT$25,000

### 適合客戶

- 遠端團隊
- 小型公司
- 需要多人共用報表的工作室
- 使用 Google Sheets 管理營運資料的團隊

## 備註

實際價格會依資料格式、欄位數量、自動化流程複雜度、是否需要 GUI、是否需要 Google API 串接而調整。
