# Examples

這個資料夾放 Excel 自動報表工具的範例輸入與輸出說明。

## 範例輸入

範例檔案：

```text
sample.xlsx
```

內容：

| 客戶 | 金額 |
| --- | ---: |
| A公司 | 1000 |
| A公司 | 3000 |
| B公司 | 2000 |

## 範例輸出

執行後會在專案的 `output/` 資料夾產生：

```text
result_YYYYMMDD_HHMMSS.xlsx
```

輸出 Excel 包含三個工作表：

- `原始資料`
- `清理後資料`
- `統計報表`

統計報表範例：

| 客戶 | 金額總和 |
| --- | ---: |
| A公司 | 4000 |
| B公司 | 2000 |
