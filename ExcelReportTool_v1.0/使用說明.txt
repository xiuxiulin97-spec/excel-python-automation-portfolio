Excel Report Tool v1.0 使用說明

本工具是 Excel 自動報表工具。
使用者不需要懂 Python，也不需要安裝 Python。

請直接使用資料夾中的 ExcelReportTool.exe。

一、如何開啟程式

1. 打開 ExcelReportTool_v1.0 資料夾。
2. 找到 ExcelReportTool.exe。
3. 用滑鼠左鍵雙擊 ExcelReportTool.exe。
4. 如果 Windows 出現安全提醒，請選擇「仍要執行」或「允許」。
5. 程式視窗開啟後，就可以開始使用。

二、如何選擇 Excel

1. 在程式視窗中，點選「選擇 Excel」。
2. 選擇要整理的 Excel 檔案。
3. 如果只是測試，可以選擇資料夾中的 sample.xlsx。
4. 選好檔案後，畫面會顯示已選擇的檔案名稱。

三、如何開始分析

1. 確認已經選擇 Excel 檔案。
2. 點選「開始分析」。
3. 等待程式處理資料。
4. 處理完成後，程式會顯示完成訊息。
5. 完成後可以到 output 資料夾查看報表。

四、結果檔在哪裡

結果檔會產生在：

output 資料夾

檔名格式會像這樣：

result_YYYYMMDD_HHMMSS.xlsx

範例：

result_20260601_152008.xlsx

產生的 Excel 報表包含三個工作表：

1. 原始資料
2. 清理後資料
3. 統計報表

五、Excel 檔案格式要求

Excel 檔案至少需要有以下兩個欄位：

1. 客戶
2. 金額

範例：

客戶,金額
A公司,1000
A公司,3000
B公司,2000

六、常見錯誤

1. 沒有選擇 Excel 檔案

請先點選「選擇 Excel」，選好檔案後再按「開始分析」。

2. 找不到「客戶」欄位

請確認 Excel 第一列欄位名稱中有「客戶」。

3. 找不到「金額」欄位

請確認 Excel 第一列欄位名稱中有「金額」。

4. 金額不是數字

請確認「金額」欄位內容是數字，例如 1000、2500、3000。
不要輸入無法計算的文字。

5. Excel 檔案打不開

請確認檔案是 .xlsx 或 .xlsm 格式。
如果檔案正在被 Excel 開啟，請先關閉後再重新執行。

6. 找不到結果檔

請查看 ExcelReportTool.exe 同一層資料夾中的 output 資料夾。
如果沒有看到 output，請重新執行一次分析。

7. Windows 阻擋程式

這是 Windows 對未簽署 EXE 的安全提醒。
如果確認檔案來源可信，可以選擇「仍要執行」。

七、注意事項

1. 本工具不會修改原始 Excel 檔案。
2. 本工具會產生新的報表檔案。
3. v1.0 固定讀取 Excel 的第一個工作表。
4. 請不要刪除 ExcelReportTool.exe。
5. 如果要重新分析，只要重新選擇 Excel 並點選「開始分析」即可。
