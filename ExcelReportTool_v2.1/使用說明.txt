Excel Report Tool v2.1 使用說明

適用對象：
完全不懂 Python 的一般使用者。

一、如何開啟程式

1. 打開 ExcelReportTool_v2.1 資料夾。
2. 雙擊 ExcelReportTool.exe。
3. 如果 Windows 出現安全提醒，請確認來源可信後選擇繼續執行。
4. 開啟後會看到「Excel Report Tool v2.1 通用表格分析工具」主畫面。

二、如何選擇 Excel / CSV

1. 點選「選擇 Excel / CSV」。
2. 選擇要整理的 Excel 或 CSV 檔案。
3. Excel 檔案會固定讀取第一個工作表。
4. 選擇成功後，畫面會顯示檔名、列數、欄位數與欄位分析結果。

三、如何選擇分析欄位

1. 分組欄位：
   選擇要分類統計的欄位，例如客戶、公司、用途、代理人、醫院名稱。

2. 統計欄位：
   選擇要加總或平均的金額欄位，例如金額、費用、成交金額、項目金額。

3. 日期欄位：
   如果資料有日期，可以選擇日期欄位產生月份統計。
   如果不需要月份統計，請選擇「不使用」。

四、如何產生報表

1. 確認已選擇檔案。
2. 確認已選擇分組欄位與統計欄位。
3. 點選「產生報表」。
4. 程式會自動清理資料並產生 Excel 報表。
5. 完成後畫面會顯示輸出路徑。

五、結果檔在哪裡

報表會產生在程式所在資料夾下的 output 資料夾。

檔名格式：

output/result_v2_YYYYMMDD_HHMMSS.xlsx

輸出 Excel 會包含：

1. 原始資料
2. 清理後資料
3. 統計報表

六、常見錯誤

1. 沒有選擇檔案
   請先點選「選擇 Excel / CSV」。

2. 沒有選擇統計欄位
   請選擇一個數字或金額欄位。

3. 統計欄位不是數字
   請改選金額、費用、成交金額等可計算欄位。

4. Excel 正在開啟
   如果輸出檔已在 Excel 中開啟，請先關閉後再重新產生報表。

5. 日期統計不正確
   請確認日期欄位格式是否一致，或先選擇「不使用」。

6. 找不到 output 資料夾
   第一次成功產生報表後，程式會自動建立 output 資料夾。

七、建議使用方式

第一次使用時，請先用 sample.xlsx 測試。
確認流程沒問題後，再使用自己的 Excel 或 CSV。

請不要把客戶真實資料截圖上傳到公開網站。
