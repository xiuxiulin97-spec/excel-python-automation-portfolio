# Excel + Python 自動化工具開發者作品集摘要

## 我的作品集定位

我的定位是：

> Excel + Python + 自動化工具開發者

我專注於使用 Python、Excel、資料清洗與自動化流程，協助中小企業、個人工作室和行政團隊減少重複整理資料的時間。AI 功能會在後續專案完成後再正式加入作品集定位。

這個作品集不是理論學習紀錄，而是以「可以實際執行、可以展示、可以接案」為目標建立的工具集合。

## 專案 1：Excel 自動報表工具

路徑：`./`

這個工具可以讓使用者選擇 Excel 檔案，自動讀取第一個工作表，清除空白列、刪除重複資料，並依照 `客戶` 欄位統計 `金額` 總和。

輸出結果是一份新的 Excel 報表，包含：

- `原始資料`
- `清理後資料`
- `統計報表`

商業價值：

- 協助店家、業務、行政或財務人員快速整理銷售資料。
- 減少人工複製貼上與手動加總。
- 適合作為 Excel 自動報表接案展示。

## 專案 2：Customer Cleaner 客戶資料清洗工具

路徑：`./customer-cleaner/`

這個工具可以讀取 Excel 或 CSV 客戶名單，自動統一欄位名稱、清理電話格式、檢查 Email、排除問題資料，並輸出乾淨客戶名單。

輸出結果是一份新的 Excel 報表，包含：

- `原始資料`
- `清理後資料`
- `問題資料`

商業價值：

- 協助中小企業整理表單、活動報名、舊客戶名單或電商名單。
- 找出 Email 錯誤、缺少姓名、缺少聯絡方式的資料。
- 適合作為資料清洗與客戶管理自動化接案展示。

## 我能解決的問題

- Excel 資料整理太花時間
- 每週或每月報表需要重複製作
- 客戶名單欄位混亂
- 電話與 Email 格式不一致
- 手動去重容易出錯
- 非技術使用者需要簡單 GUI 工具
- 小公司需要低成本的流程自動化

## 適合的客戶類型

- 中小企業
- 個人工作室
- 電商賣家
- 業務團隊
- 行政與財務人員
- 活動主辦方
- 顧問、課程、服務型團隊
- 需要整理 Excel 或 CSV 的接案客戶
